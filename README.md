[Android CI](https://github.com/Wiktor-coder/DebitCredit/actions/runs/28165677731/job/83416926296)

## ✨ Особенности

- 📈 Отслеживание расходов по категориям
- 💰 Добавление доходов
- 📊 Графики и статистика
- 🌍 Поддержка русского и английского языков
- 🌙 Тёмная тема
- 💲 Актуальный курс валют ЦБ РФ на экране настроек

## 🛠️ Технологии

- Kotlin
- Android Jetpack (ViewModel, Navigation, Room)
- Coroutines
- CustomBarChart an StatsView

## 📥 Установка

Скачайте последний APK из [Releases](https://github.com/Wiktor-coder/DebitCredit/releases)

## 🏗️ Сборка из исходников

```bash
./gradlew assembleDebug
```
